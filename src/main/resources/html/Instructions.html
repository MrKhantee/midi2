Instructions

This application allows you to input a sequence of notes. Here are some of the features and how to use them:

  • Input notes - You have two options: either use the note buttons or type the letters on your keyboard.
  • Add rests and continued notes - To add a rest, use Command-R or SPACE. To continue the previous note, use Command-C.
  • Edit existing notes:
    • Delete one note - Command-D
    • Delete all notes - Shift-Command-D
  • Playback:
    • Play - Command-P
    • Stop playing - Command-T
  • Files:
    You can save your MIDI sequences as files so that you can open them back up and play them later.
    • Save - Command-S
    • Open - Command-O
